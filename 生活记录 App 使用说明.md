# 生活记录 App 使用说明

## 一、App 简介

生活记录是一个纯本地的个人日记/记录 App，帮助你按时间线记录日常生活中的点滴。支持自定义树状标签体系、属性标签、项目归类、文本格式、AI 助写、批量管理、模板复用、打卡追踪、主题切换、Excel 导出和备份恢复。

> **数据安全**：所有数据存储在手机本地（AI 助写除外）。可通过 JSON 导出+坚果云 WebDAV 实现多设备间手动同步。

---

## 二、首页（时间线）

打开 App 后看到的是**首页时间线**：

- 记录按**日期分组**显示，每天为一个卡片
- 点击日期标题行可展开/收起当天的记录
- 每条记录显示：**时间 + 标题/内容（最多 2 行）+ 项目名 + 树状标签 + 属性标签**
- 点击记录 → 进入**详情页**查看全文

### 顶部工具栏

**第一行（AppBar）**：

| 图标 | 功能 |
|------|------|
| 📋 | 进入**列表视图** |
| 🏷 | 进入**标签管理** |
| ✨ | 进入**属性标签管理** |
| ✅ | 进入**打卡** |
| 📁 | 进入**项目管理** |
| ⋮ | 更多菜单：模板管理 / AI 总结 / 导出 Excel / 数据备份 / 坚果云同步 / 切换入口 / AI 助写设置 / 主题 / 隐私锁 |

**第二行（展开/折叠栏）**：仅在有时记录显示。左侧「展开」「折叠」按钮控制所有日期分组的展开/折叠状态，右侧有 **↪ 撤销按钮**（灰色禁用，有操作时高亮可点）和 **↕ 排序按钮**（切换正序/倒序），以及记录总数。

### 底部 ⊕ 按钮

点击新建一条记录。

## 三、添加 / 编辑记录

### 新建记录

点击首页右下角的 **⊕** 按钮，弹出编辑器：

1. **从模板导入**（可选）— 点击顶部紫色边框区域，选择已有模板自动填充所有字段
2. **点击「事项简介」** — 弹出输入框，支持多行（标题会加粗显示）
3. **点击「详细情况」** — 弹出大输入框，顶部有**格式化工具栏**
4. **选择树状标签**（可选）— 在标签树中选一条路径（单分支）
5. **选择属性标签**（可选）— 多选扁平标签
6. **选择项目**（可选）— 单选
7. **填写对接人**（可选）— 点击输入对接人姓名
8. **填写后续待办**（可选）— 点击输入后续待办事项
9. **另存为模板**（可选）— 将当前填好的内容保存为模板
10. 点击右上角 **「保存」**

### 格式化工具栏

内容输入框顶部的工具栏按钮：

| 按钮 | 插入语法 | 效果 |
|------|---------|------|
| **B** | `**文字**` | **粗体文字** |
| *I* | `*文字*` | *斜体文字* |
| • | `- 项目` | 无序列表 |
| 1. | `1. 项目` | 有序列表 |
| T | `# 标题` | 标题 |
| ✨ | （弹出风格菜单） | AI 润色 |

### AI 助写

点击 ✨ 按钮 → 选择写作风格 → AI 自动润色当前文本。支持以下风格：

| 风格 | 说明 |
|------|------|
| 用户原文风格 | 只修正语病，保留原味 |
| 正式书面风格 | 规范化书面语 |
| 跟自己聊天风格 | 口语化表达 |
| 鲁迅风格 | 冷峻凝练 |
| 徐志摩风格 | 优美诗意 |
| 文言文风格 | 翻译为文言文 |
| 自定义 | 用户自定提示词 |

AI 供应商和 API Key 在 `⋯` → **AI 助写设置** 中配置。

### 编辑已有记录

在首页或列表视图中点击记录 → 进入**详情页** → 右上角 ✎ 按钮 → 修改后保存。

## 四、详情页（卡片式展示）

页面分为 5 张卡片：

1. **元信息卡** — 时间、项目、树状标签、属性标签
2. **对接人卡** 👤 — 显示对接人姓名（如果有）
3. **事项简介卡** — 标题文字
4. **内容卡** — 完整内容，格式已渲染
5. **后续待办卡** ✅ — 显示后续待办（如果有）

右上角 ✎ → 编辑，📑 → 存为模板。

---

## 五、标签管理

### 树状标签

标签是**树状结构**，支持无限层级。例如：

```
生活
├── 饮食
│   ├── 早餐
│   ├── 午餐
│   └── 晚餐
├── 运动
│   ├── 跑步
│   └── 健身
└── 阅读
```

**操作**：展开/收起、添加子标签、重命名、删除、拖拽排序、移动层级。

### 属性标签

扁平的属性标签体系，可按分组归类。用于标记心情、地点、优先级等维度。

## 六、项目管理

项目用于给记录归类。项目可以分组，一条记录通常对应一个项目。

- **添加项目/分组**：点击右下角 ⊕
- **重命名/删除**：点击项目右侧的 ⋮

---

## 七、列表视图

列表视图提供**筛选**、**分组**和**批量操作**功能。

### 筛选（AND 组合多选）

| 筛选维度 | 说明 |
|---------|------|
| 📅 日期 | 选择单日或时段 |
| 🏷 树状标签 | 多选，OR 逻辑 |
| ✨ 属性标签 | 多选，OR 逻辑 |
| 📁 项目 | 多选，OR 逻辑 |
| 👤 对接人 | 输入关键词，显示已有对接人列表供点选 |
| ✅ 有后续待办 | 开关，只显示有待办事项的记录 |
| 🔍 关键词 | 搜索标题、内容、标签名 |

所有筛选条件用 **AND** 组合，同维度内用 **OR**。

### 分组

筛选栏右侧选择分组方式：按日期 / 按项目 / 按树状标签 / 按属性标签 / 不分组。

### 批量操作

1. 点击右上角 ✓ 进入选择模式
2. 勾选记录
3. 工具栏操作：替换树状标签 / 替换属性标签 / 设置项目 / 批量删除

---

## 八、模板功能

### 模板管理

首页 `⋯` → **模板管理**，可查看、创建、编辑、删除模板。

模板包含：名称、事项简介、详细情况、树状标签、属性标签、项目、对接人、后续待办。

### 从模板导入

新建记录时，编辑器顶部有 **「从模板导入」** 按钮：
1. 点击 → 弹出模板列表
2. 选择模板 → 自动填充所有字段
3. 可在此基础上自由修改再保存

### 另存为模板

- **编辑器中**：字段输入区下方有 **「另存为模板」** 按钮
- **详情页**：AppBar 上的 📑 图标按钮

---

## 九、打卡功能

首页 AppBar 点击 ✅ 图标进入打卡页面。

### 打卡事项管理

打卡页面右上角 ⚙ → **管理事项**，可添加/编辑/删除打卡事项。

每个打卡事项可关联树状标签、属性标签和项目，打卡时自动附加。

### 打卡操作

- 事项以 3 列网格展示
- 点击未打卡的事项 → ✅ 完成打卡（自动生成一条记录到时间线）
- 已打卡的事项显示绿色勾选状态
- 每个事项显示连续打卡天数 🔥

### 统计分析

| 统计项 | 说明 |
|--------|------|
| 今日概览 | 今日已打卡数 / 总事项数 |
| 本周趋势 | 7 天柱状图，每天打卡数 |
| 最长连续 | 所有事项中最长连续打卡天数 |
| 月度热力图 | 当月每天打卡数，颜色深浅表示密度 |
| 完成率排名 | 各事项本月打卡天数及进度条 |

## 十、AI 总结

首页 `⋯` → **AI 总结**，选择日期范围后生成总结。

**默认分析方向**：
1. **做了什么** — 概括主要事项，尽量包含量化数据
2. **分析重点** — 关键事项、重复出现的主题或模式
3. **后续待办** — 还需要跟进或处理的事项

仅基于已有记录分析，不引申或推测记录之外的内容。

可在 **「总结要求」** 输入自定义提示词定制分析方向。

## 十二、隐私锁

首页 `⋯` → **隐私锁** → 开启后可设置：

| 设置项 | 说明 |
|--------|------|
| 启用隐私锁 | 开/关 |

## 十四、坚果云同步

首页 `⋯` → **坚果云同步**，可配置 WebDAV 在多设备间手动同步数据。

### 第一次配置

1. 在坚果云官网 → 安全选项 → **添加第三方应用密码**
2. App 内填写：**WebDAV 地址**、**邮箱**、**应用密码**
3. 点击 **「测试连接并保存」**

### 上传到云端

点击 **「上传到坚果云」**，将当前所有数据导出为 JSON 并上传。

### 从云端恢复

1. 点击 **「刷新云端备份列表」** 查看云端文件
2. 选择需要恢复的备份 → 点击 **「恢复」**
3. ⚠️ 恢复会替换本地所有数据，不可撤销！

> 当前同步为手工备份/恢复模式，每次恢复为全量替换。

## 十八、常见问题

**Q: 数据存在哪里？**
本地 SQLite 数据库。AI 助写会发送文本到第三方 API。

**Q: AI 需要什么 Key？**
推荐 DeepSeek（platform.deepseek.com），也支持通义千问、Claude、OpenAI 等。

**Q: 打卡记录会出现在时间线吗？**
会的。每次打卡自动生成一条带 ✅ 标记的记录，在首页时间线中可见。
